Memory Pada Komputer

Memory adalah perangkat atau sistem yang digunakan untuk menyimpan informasi untuk penggunaan langsung dalam komputer atau perangkat keras komputer dan perangkat elektronik digital yang terkait.

MEMORI HIRARKI:
Memori Hirarki memisahkan penyimpanan komputer menjadi hierarki berdasarkan waktu respons. Karena waktu respons, kompleksitas, dan kapasitas terkait, levelnya juga dapat dibedakan berdasarkan kinerja dan teknologi pengontrolannya. Memori Hirarki mempengaruhi kinerja dalam desain arsitektur komputer, prediksi algoritma, dan konstruksi pemrograman tingkat rendah yang melibatkan lokalitas referensi
Ada empat tingkat penyimpanan utama:
• Internal, Prosesor mendaftar dan cache.
• Utama, sistem RAM dan kartu pengontrol.
• Penyimpanan massal on-line, Penyimpanan sekunder.
• Penyimpanan massal off-line, Penyimpanan tersier dan Off-line

JENIS JENIS MEMORY:
• Memory Read Only (ROM)
• Memory Read/Write
• Static Memory
• Dynamic Memory
• Volatile
• Non-volatile
